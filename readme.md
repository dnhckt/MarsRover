# Mars Rover Problem

Takes user input for movement of two mars rovers within a given plateau. 

Files:
    * Main.java
    * Rover.java
    * Motor.java
    * IntPair.java

## How To Run

```bash

javac *.java && java Main

```

## Example

Sample Input:

```bash

5 5

1 2 N

LMLMLMLMM

3 3 E

MMRMMRMRRM

```

Expected Output:

```bash

1 3 N

5 1 E

```
